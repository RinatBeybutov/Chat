# Краткая информация

Это приложение - городской чат, в котором пользователи могут
писать сообщения

- Только один чат, приватных комнат и других каналов не
  требуется
- Может быть несколько участников
- И список тех, кто онлайн

# Реализовано

- маппинг
- добавить сущность комментариев
- написать фронт с отображением последних 10 комментариев
- написан один юнит и один интеграционный тест
- написана базовая авторизация
- добавлена регистрация
